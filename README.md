# SoundShare Web App Frontend

## Built with
- Vue 3
- Tailwind CSS
- Vite

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
